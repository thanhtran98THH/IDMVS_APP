针对C# demo的运行：

若已执行MvCodeCameraSDK_VX.X.X.X_VC90_Runtime.exe文件，只需要拷贝SDK/C#路径下对应MvCodeReaderSDK.Net.dll放置c#demo的可执行exe路径下， 即运行demo的exe;

注意：若未执行Runtime包文件，则需拷贝SDK/C/dll路径下win32或win64文件下的dll和ini配置文件和SDK/C#路径下对应平台MvCodeReaderSDK.Net.dll，放置对应C#Demo的可执行文件exe路径下，即可运行demo的exe；


【特殊说明】
C#demo的可执行文件exe路径下已包含MvCodeReaderSDK.Net.dll，即C#封装的dll库，C#demo依赖该MvCodeReaderSDK.Net.dll（区分win32/win64）。


For the operation of the c# demo:

If executed MvCodeCameraSDK_VX.X.X.X_VC90_Runtime.exe files, can be directly run the demo exe;

Note:
If the Runtime file has not been executed, to run the demo EXE file, you should copy the DLL and INI configuration files from the directory 
SDK/C/dll/win32 or SDK/C/dll/win64 to the directory of C#demo EXE file.

Note:
C# demo depends on the encapsulated DLL library MvCodeReaderSDK.Net.dll which already exists in the directory of 
C# demo EXE file. You should note the 32-bit and 64-bit version.
